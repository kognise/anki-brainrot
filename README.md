# [Anki Brainrot](https://ankiweb.net/shared/info/1331347293)

My opinionated, lightweight gamification plugin for Anki, featuring:

- 🎊 Confetti on "Good" and "Easy" answers
- ➡️ Progress bar
- ✅ Animated confirmation popups
- 🔔 Sound effects
- 🐶 Out of the box configured puppy reinforcement
- 🎨 Nicer looking buttons

Everything is toggleable, but only the deck completion sound is configurable to whatever your brain has the biggest dopamine response to. I'll probably add more features in the future.
